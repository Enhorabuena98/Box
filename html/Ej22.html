Documentación del Código: VentanaPrincipal

Esta página describe la funcionalidad y estructura del código de la clase VentanaPrincipal.

Introducción

La clase VentanaPrincipal implementa una aplicación gráfica utilizando PySide6. Proporciona menús, herramientas y funcionalidades avanzadas como edición de texto, cambios de estilo y acciones interactivas.

Estructura del Código

Módulos Importados

El código utiliza los siguientes módulos:

  • os: Gestión de rutas de archivo.
  • PySide6.QtWidgets: Para construir la interfaz gráfica (QMainWindow, QAction, etc.).
  • PySide6.QtGui: Para configuraciones gráficas y atajos de teclado.
  • PySide6.QtCore: Soporte para constantes y señales Qt.
  • PySide6.QtWebEngineWidgets: Para integrar vistas web dentro de la aplicación.

Atributos y Métodos Principales

La clase VentanaPrincipal se estructura en las siguientes partes:

  • Atributos:
    • self.status_bar: Barra de estado para mensajes informativos.
    • self.dockText: Área de texto anclada para editar contenido.
    • self.contador_palabras_label: Contador de palabras dinámico.
  • Métodos:
    • crear_accion(): Crea acciones configuradas con íconos, atajos y eventos.
    • contar_palabras(): Actualiza el contador de palabras en el área de texto.
    • Acciones relacionadas con la edición de texto (cortar(), copiar(), etc.).
    • Acciones relacionadas con estilos y configuraciones (cambiar_fuente(), cambiar_color()).
    • Acciones de integración web (abrir_chatgpt(), abrir_lofi(), etc.).

Funcionalidad

Menús y Herramientas

El programa implementa menús y una barra de herramientas con las siguientes opciones:

  • Menú Archivo: Contiene acciones como abrir, guardar, ayuda y salir.
  • Menú Edición: Ofrece opciones para cortar, copiar, pegar y gestionar el texto.

Integración Web

Se incluyen funcionalidades para abrir servicios web directamente desde la aplicación:

  • ChatGPT: Abre el sitio de ChatGPT.
  • Lofi: Accede a una página de música Lofi.
  • River: Muestra documentación de PyQt.

Ejemplo de Código

A continuación, se muestra un ejemplo de cómo iniciar la aplicación:

from PySide6.QtWidgets import QApplication
from VentanaPrincipal import VentanaPrincipal

app = QApplication([])
ventana = VentanaPrincipal()
ventana.show()
app.exec()

Conclusión

Este código es un ejemplo avanzado de una aplicación PySide6 que combina edición de texto, integración web y una interfaz gráfica intuitiva.
